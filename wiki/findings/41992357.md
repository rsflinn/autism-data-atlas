# Causal effects of single-carbon metabolism and vitamin levels on autism spectrum disorder risk: a bidirectional Mendelian randomized study

## Key Findings
Bidirectional Mendelian randomization using published GWAS summary statistics to test whether one-carbon metabolism components and vitamin levels causally influence ASD risk. Abstract text was not available in the PubMed record, so specific effect estimates were not extracted.

## Relevance to Atlas
Uses GWAS-based instrumental variables, consistent with the atlas's focus on common variant contributions to autism liability. Metabolic-nutrient pathway, not regulatory variant pathway, so relevance is peripheral.

## Genes
None in atlas 49-gene set

## Methods
Two-sample bidirectional Mendelian randomization on GWAS summary statistics.
