# Genetic overlap and shared risk loci between autism spectrum disorder and cardiometabolic traits.

## Key Findings
GWAS pleiotropy analysis finds 100 shared loci between ASD and cardiometabolic traits using MiXeR + pleioFDR; polygenic overlap despite near-zero trait-level correlation.
