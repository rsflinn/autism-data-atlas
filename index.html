Getting Started with Autism Genetics Data

A practical guide for citizen scientists, parents, students and independent researchers

What You're Looking At

Autism genetics data comes in several flavors, each answering a different biological question:

  • Gene expression data (RNA-seq) measures which genes are turned on in which cells. A typical RNA-seq experiment produces a spreadsheet where rows are genes and columns are individual cells or tissue samples, with values representing how much mRNA each gene produced.
  • Variant data (WGS/WES) catalogs differences in DNA sequence between individuals. Think of it as a diff file: positions where your DNA differs from the reference genome, what the difference is, and how common it is in the general population.
  • GWAS summary statistics test whether common genetic variations are more frequent in people with autism than in controls. The output is a table of SNP positions with p-values and effect sizes.
  • Regulatory networks map how genes control each other. These come from ChIP-seq (which transcription factors bind where), ATAC-seq (which DNA regions are accessible), or computational inference (which genes likely regulate which).

    • File Formats You'll Encounter

    VCF (Variant Call Format)

    A text file listing every place someone's DNA differs from the reference genome. Each row = one variant. Columns = chromosome, position, reference allele, alternate allele, quality scores, and sample-level information. Opens with pysam in Python or bcftools on the command line.

    H5AD

    The standard format for single-cell RNA-seq data. Stores a matrix where rows = cells and columns = genes, plus metadata about each cell (cell type, donor, condition, etc.). Opens with scanpy in Python. Usually compressed to gigabyte-scale files.

    FASTQ

    Raw sequencing reads from the sequencer. You almost never need these unless you're reprocessing data from scratch. Each read is 4 lines: identifier, sequence, separator, and quality scores. Files are huge (10–100 GB per sample).

    GZ / BGZ

    Compressed files. Most genomic data lives compressed. Python's gzip library or pandas can read them directly without decompressing to disk.

    TSV / CSV

    Tab or comma separated text. The friendliest format. Opens in Excel, Google Sheets, pandas, R, or any text editor. Most supplementary tables from papers arrive as TSV or CSV.

    BED

    Genomic coordinates. Three columns minimum: chromosome, start position, end position. Used for regulatory elements, peaks, regions of interest. Often extended with additional columns for scores or metadata.

    XLSX

    Excel files. Common for supplementary tables in papers. Opens with openpyxl or pandas in Python.

    Setting Up Your Environment

    You need Python 3.9 or later. Install a few essential packages:

    pip install pandas numpy scanpy anndata openpyxl GEOparse requests

    For variant analysis, add:

    pip install pysam pyliftover vcf

    For plotting:

    pip install matplotlib seaborn plotly

    For interactive exploration, we recommend Jupyter notebooks. If you want zero setup, use Google Colab.

    Your First Analysis

    Let's walk through downloading a real dataset from GEO: GSE28521, a microarray study comparing cortical gene expression in people with and without autism. It's fully public, small enough to handle on a laptop, and published in a peer-reviewed journal.

    import GEOparse import pandas as pd # Download the dataset gse = GEOparse.get_GEO('GSE28521', destdir='./data') # List the samples for sample in gse.metadata['sample_ids']: print(f"{sample}: {gse.gsms[sample].metadata['title']}") # Extract expression matrix (rows = genes, columns = samples) expression = gse.pivot_samples('VALUE') print(expression.shape) # (number of genes, number of samples) # Load phenotype data phenotypes = pd.DataFrame({ sample: gse.gsms[sample].metadata for sample in gse.metadata['sample_ids'] }).T

    Now you have the expression data and sample metadata. You could compare gene expression between autism and control samples, look for patterns using dimensionality reduction, or cross-reference your findings with genetic databases.

    Access Levels Explained

    PUBLIC

    Download immediately, no barriers. Start here. GEO, Zenodo, and most bioRxiv preprints fall into this category.

    REGISTERED

    Create a free account (GEO, Synapse, Zenodo). Takes 5 minutes. Minor friction, but data is still freely accessible to anyone.

    CONTROLLED

    Requires institutional affiliation and an approved Data Use Agreement (DUA). Timeline: weeks to months. Protects participant privacy for sensitive datasets like patient cohorts. NIH dbGaP and the Simons Simplex Collection work this way.

    GATED

    Available "upon reasonable request" to the corresponding author. Success rate varies. Email them politely with your affiliation and research plan. This is where independent researchers sometimes hit a wall.

    • About This Project

    This resource was built by Ryan Flinn, a journalist and parent of a child with profound autism and drug-resistant epilepsy. Over 130+ working sessions beginning in early 2025, Flinn independently reanalyzed his daughter's genetic data using publicly available research datasets and AI-assisted analysis tools.

    The biggest recurring barrier wasn't the analysis itself — it was finding the data. Datasets are scattered across GEO, Synapse, Zenodo, journal supplements, and institutional portals with different access requirements, file formats, and naming conventions. This catalog exists so other parents, citizen scientists, graduate students, and independent researchers don't have to repeat the scavenger hunt.

    The gene network explorer visualizes how autism-associated genes interact across biological layers, from regulatory factors at the top through transcription factors and ion channels down to clinical phenotype. It's built from published research data and is intended as a learning tool, not a diagnostic instrument.

    This is an open-source project. The dataset catalog lives as a JSON file on GitHub. Anyone can submit a pull request to add a dataset, correct an error, or update access information.
